### universeterminal
the **universeterminal** is a CLI emulator web application that allows you to explore a custom filesytem designed to simulate the scale of the universe. 

### Supported Commands
| Command     | Description                                |
|-------------|--------------------------------------------|
| `help`      | List commands                              |
| `clear`     | Clear terminal history                     |
| `bigbang`   | Reset the universe                         |
| `ls`        | List files and directories                 |
| `tree`      | Display explored universe                  |
| `pwd`       | Print working directory                    |
| `cd`        | Change directory                           |
| `exit`      | Exit the terminal and return to homepage   |
